# Project Structure Guide

## Project Organization
This document serves as a developer-facing guide for the organization and file structure of the Comora project.

## File Structure
- `src/`: Contains the source code of the application.
- `tests/`: Contains all the test cases and testing scripts.
- `docs/`: Contains documentation files including this guide.
- `assets/`: Contains images, CSS files, and other static resources.
- `scripts/`: Contains scripts for build, deployment, and setup.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Aditya0265/Comora.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Comora
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Feature Tracking
All features and enhancements are tracked in the Issues section of the repository. Please ensure to create an issue for any new feature or bug.

For a detailed understanding of the system requirements, refer to the Software Requirements Specification (SRS) v1.2 document.
